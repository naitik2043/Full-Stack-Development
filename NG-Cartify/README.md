# 🛒 NG-Cartify – React E-Commerce Cart

A modern and responsive shopping cart web application built using **React.js** that simulates real-world e-commerce cart functionality with dynamic state management and reusable components.

## 📌 Features

✨ Browse products  
🛍️ Add items to cart  
❌ Remove items from cart  
🔢 Update product quantity  
💰 Real-time total price calculation  
⚡ Fast and dynamic UI updates  
📱 Fully responsive design  
🧩 Reusable component architecture
